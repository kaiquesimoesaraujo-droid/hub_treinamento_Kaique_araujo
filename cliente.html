Cliente

Módulo voltado ao tratamento das notas de cliente, com foco na integração com o SAC, leitura dos retornos da planilha e definição da tratativa fiscal correspondente.

Neste módulo

Notas de ajuste Integração com SAC Tabela de referência Matriz de decisão Processo de atualização

Notas de ajuste (estornos)

Para notas de ajuste, a orientação segue a mesma lógica aplicada ao fluxo de fornecedor: não manifestar a nota de ajuste. Deve-se localizar a NFD referenciada, realizar a manifestação correspondente e concluir o ajuste na lista de controle.

Integração com SAC via planilha

Responsável pelo preenchimento Equipe de SAC.
Campos preenchidos Status da nota e número do ticket no arquivo de integração.

Para notas de devolução, a tratativa fiscal depende do retorno informado pelo SAC. Por isso, o preenchimento do campo Status deve seguir exatamente os valores padronizados definidos para a planilha.

Tabela de referência para preenchimento do Status

A tabela abaixo apresenta os valores válidos para o campo Status e o respectivo significado operacional de cada opção.

Importante: esta tabela é apenas uma referência de apoio para preenchimento correto da planilha. Ela não é enviada ao SAC e não substitui o campo de seleção utilizado no arquivo.
Status Descrição
NFD validada - Transporte NFD foi validada e está aguardando coleta.
NFD validada - Logística NFD foi validada e está aguardando solicitação de coleta.
NFD não validada NFD não foi validada pelo time SAC.
NFD sem protocolo Não foi encontrado protocolo referente à NFD.
  • Utilize apenas os valores padronizados apresentados nesta referência.
  • O preenchimento incorreto impacta a leitura do fluxo automático no SharePoint.
  • A tabela serve como apoio visual e orientação operacional.

Processo de atualização

  1. Geração da planilha
    O sistema gera o arquivo de integração para preenchimento pelo SAC.
  2. Preenchimento pelo SAC
    O time informa status, ticket e dados relacionados ao atendimento.
  3. Leitura automática do ID
    O fluxo do SharePoint identifica o ID da nota no arquivo.
  4. Atualização da lista principal
    A lista é atualizada automaticamente com base no retorno recebido.
